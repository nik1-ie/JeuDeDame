# Projet Multidames
Par Stephany Santos Feirera de Sousa et Niekita Joseph
TP 12 - DM 14

# Comment jouer
Ouvrir un terminal dans le dossier et taper 
clang multi_dames.c -std=c17 -o multidames
Ce fichier sera exécuté selon les tests fourni par Platon C.


Le fichier multi_dames_manuel.c correspond au jeu manuel, jouable sur terminal - il suffit de suivre les instructions affichées sur le terminal, exécutés de la même façon.
